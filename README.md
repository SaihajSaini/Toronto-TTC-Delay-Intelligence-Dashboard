# Toronto TTC Delay Intelligence Dashboard 🚇

## Overview
This project analyzes TTC subway delay data to identify patterns, trends, and factors contributing to service disruptions.

## Objectives
- Identify the most delayed subway lines and stations
- Analyze delay patterns by time and day
- Discover common causes of delays
- Create an interactive dashboard for insights

## Tools Used
- Python (Pandas, NumPy)
- SQL (PostgreSQL)
- Tableau
- Git/GitHub
